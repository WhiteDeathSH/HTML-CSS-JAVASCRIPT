Definition:

  • HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page.

  • HTML is used to provide structure to a webpage and make it accessible to users of the internet through text, visual formatting and search factors.

    • Sample of Program

    <!DOCTYPE html>
    <html>
    <head>
    <title>My First HTML Page</title>
    </head>
    <body>
    <h1>Hello World!</h1>
    <p>This is a Sample Paragraph</p>
    </body>
    </html>

    Output of the Program

    Hello World!

    This is a Sample Paragraph
